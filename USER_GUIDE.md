## ‚è∞ Managing Time Cards

### Viewing Your Current Time Card

1. **From Dashboard**: Click "Current Time Card"
2. **From Menu**: Navigate to "Time Cards" ‚Üí "Current"

Your time card shows:
- Pay period dates (bi-weekly)
- Daily time entries
- Total hours worked
- Submission status

### Adding Time Entries

**Daily Time Entry:**
1. Click "Add Entry" or tap a date
2. Select the date
3. Choose day type:
   - **Regular**: Normal work day
   - **Sick**: Using sick time
   - **Vacation**: Using vacation time
   - **Personal**: Personal day
   - **Professional Development**: Training/meetings

4. Enter times:
   - **Time In**: When you arrived
   - **Time Out**: When you left
   - **Break Time**: Total break minutes (optional)

5. Add notes if needed
6. Click "Save Entry"

**Tips:**
- Times automatically calculate hours worked
- Break time is subtracted from total hours
- You can edit entries until the time card is submitted

### Submitting Your Time Card

1. **Review All Entries**: Make sure all days are complete
2. **Click "Submit for Approval"**
3. **Confirm Submission**: Once submitted, you cannot edit
4. **Wait for Approval**: Your manager will review and approve

**Important:**
- Submit by end of day Friday for each pay period
- Late submissions may delay payroll processing
- You'll receive email notifications about status changes

### Time Card History

View past time cards under "Time Cards" ‚Üí "History":
- See all previous pay periods
- Download/print approved time cards
- Review approval comments

## üèñÔ∏è Requesting Time Off

### Creating a Time Off Request

1. **Navigate to Time Off** ‚Üí "Request Time Off"
2. **Select Request Type**:
   - **Sick**: Illness or medical appointments
   - **Vacation**: Planned time off
   - **Personal**: Personal matters
   - **Bereavement**: Family emergencies
   - **Jury Duty**: Legal obligations

3. **Choose Dates**:
   - Start date and end date
   - The system calculates work days automatically
   - Avoid weekends and holidays

4. **Add Details**:
   - Reason (optional but recommended)
   - Expected duration
   - Any special notes

5. **Submit Request**:
   - Click "Submit Request"
   - Your manager receives immediate notification

### Tracking Request Status

**Request Statuses:**
- üü° **Pending**: Waiting for manager approval

- ‚ùå **Denied**: Request denied, see manager comments

**Check Status:**
- Dashboard shows recent requests

- Email notifications for status changes

### Time Off Calendar

View team time off at "Time Off" ‚Üí "Calendar":

**Available to:**

## üÜò Common Issues & Solutions

### Login Problems

**Forgot Password:**
1. Click "Forgot Password" on login page
2. Enter your email address
3. Check email for reset link
4. Follow instructions to create new password

**Account Locked:**
- Contact your administrator
- Usually unlocks after 15 minutes
- Multiple failed attempts cause temporary lockout

### Time Card Issues

**Can't Edit Time Card:**
- Check if already submitted
- Contact manager if approved card needs changes
- Time cards lock after submission

**Missing Time Entry:**
- Add entries for each work day
- Include sick/vacation days with appropriate type
- Don't leave days blank

**Wrong Hours Calculated:**
- Check time in/out format (AM/PM)
- Verify break time entered correctly
- Times should be in 15-minute increments

### Time Off Request Problems

**Request Denied:**
- Read manager's comments
- Discuss with manager if needed
- Resubmit with changes if appropriate

**Can't Select Dates:**
- Check for existing approved requests
- Avoid past dates
- Some dates may be blackout periods

### Mobile Issues

**App Won't Load:**
- Check internet connection
- Clear browser cache
- Refresh the page
- Try different browser

**Buttons Too Small:**
- Zoom in using pinch gesture
- Rotate to landscape mode
- All buttons meet accessibility standards
